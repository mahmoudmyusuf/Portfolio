Project Overview

The Control Room Live Presentation project is designed to provide a dynamic and automated operational overview for the control room team. It unifies data from multiple sources into a continuously updating PowerPoint presentation. The project builds on three previous implementations:

  • Automated Data Extraction from Excel – Extracts and processes field reports.
  • Field Data Dashboard – Displays live plant data from PI System.
  • Permit to Work Visualization – Manages and visualizes active work permits.

This integration provides real-time insights into well production rates, plant operations, permit statuses, and HSE updates through a single, automated presentation that updates itself when restarted.

Resources

The project gathers data from multiple sources to ensure a complete operational view:

  • Designation board – Operations team roles & responsibilities (Excel workbook).
  • Daily reports – Field data collected in Excel workbooks.
  • Live process data – Real-time transmitter values from the PI System.
  • Work Permits Database – Permit statuses stored in an Access database.
  • Multimedia elements – Training and safety videos:
    • Company intro video (Intro_Video.MP4)
    • Barrier Model training (Barrier_Model.MP4)
    • Rotating HSE/Technical learning videos (Used_Video.MP4, updated every 12 hours)

Tools & Technologies

Multiple technologies are used to automate data processing and enhance visualization:

  • PI System (PI-Link add-in) – Extracts live transmitter readings into Excel.
  • Microsoft Excel – Used for data extraction, transformation, and visualization.
  • Excel VBA – Automates data updates, file management, and PowerPoint integration.
  • Microsoft Access – Stores and retrieves permit-to-work data.
  • Microsoft PowerPoint – Displays real-time data in an automated slideshow.

Data Integration Approach

The project consolidates multiple Excel VBA modules from previous implementations into a single workbook.

1. Automated Report Data Extraction
  • Uses the (Get_rest_files) module to scan the reports folder and check:
    • Report names and timestamps to identify updates.
    • New files that need to be processed.
  • Updates workbook links using (Update_WB_links) module.
  • Processed data is used to generate:
    • One slide for sub-sea layout – Displays well production rates, phase contributions, and key highlights (e.g., well start-up/shut-in events).
    • Two slides comparing key production figures:
      • Gas feed vs. export
      • Condensate/Aqueous volumes vs. storage levels
2. Permit to Work (PTW) Data Processing
  • Runs the (PTW) module to extract permit data from the Access database.
  • Distributes data across five slides:
    • Summary of active permits by type.
    • Permit locations mapped to plant sections.
    • SIMOPS (Simultaneous Operations) risk areas.
    • Isolation and hazard details.
    • High-density work zones for safety supervision.
3. Live Field Transmitters Data Integration
  • Extracts data from PI System when the workbook opens.
  • Populates a plant overview slide showing:
    • Current production rates for gas, condensate, and water.
    • Rate changes compared to the previous hour.
    • Last 24-hour operational highlights.
4. Embedded Training & HSE Videos
  • Links three videos to run automatically:
    • Intro_Video.MP4 – Plays at the start of the presentation.
    • Barrier_Model.MP4 – Plays after permit data review.
    • Used_Video.MP4 – A rotating HSE or technical training video.
  • Uses the (used_videos) module to rename videos in a designated folder every 12 hours, ensuring a new training video plays at each cycle.
5. Designation Board
  • Manual entry Excel sheet listing operations team members and their responsibilities.

Reporting & Presentation Flow

The final PowerPoint presentation follows a structured sequence and loops continuously. When it reaches the first slide, all data links are refreshed.

  • Slide 1: Company Logo & Intro Video
  • Slides 2-3: Operations Team Designation Board
  • Slide 4: Sub-sea Layout – Well Contributions & Highlights
  • Slide 5: Plant Overview – Live Process Data
  • Slides 6-7: Report Data Comparisons (Gas, Condensate, Storage)
  • Slides 8-12: Permit to Work Overview
  • N Slides: HSE Life-Saving Rules & Safety Insights
  • Final Slides: Barrier Model & Rotating Learning Video
