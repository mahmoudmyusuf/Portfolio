Project Overview

The Permit to Work Visualization project aims to enhance the tracking and management of work permits for field activities. Daily permit data is recorded in an ACCESS database, categorizing permits into six types based on criticality and scope:

  • Confined Space Entry (CSE)
  • Hot Work (HW)
  • Break of Containment (BoC)
  • Spark Potential (SP)
  • Cold Work (CW)
  • Excavation (EXC)

Special attention is given to SIMOPS (Simultaneous Operations) permits, requiring additional controls for safe execution.

Resources

  • Permit records from ACCESS Database
  • Google Maps for site visualization

Tools & Technologies

  • Microsoft Access for permit data entry
  • Microsoft Excel for data gathering, transformation, and visualization
  • Microsoft PowerPoint for real-time presentation

Data Integration Approach

The processing plant is divided into sections and subsections based on facility locations and operations. Key enhancements include:

1. ACCESS Database Enhancements
  • Added a new table defining 4 sections with 87 subsections.
  • Implemented a filtered query to display only relevant subsections for the selected section, improving data entry efficiency.
  • Created a report-query to extract permit data based on type, status, isolation requirements, and cross-referenced permits.
2. Excel Workbook for Visualization

A new Excel workbook with two sheets (Report and Layout) was created, both linked to the ACCESS database.

Sheet 1: Report
  • Sorted permits by section and ranked them by criticality: CSE → HW → BoC → SP → CW → EXC.
  • Generated a statistics page displaying:
    • Number of permits per area
    • Permit types and distribution per discipline
  • Created three pages of live permit details, including:
    • Permit number
    • Color-coded permit types
    • Discipline
    • Task description
    • Isolation details
    • Corrective and preventive maintenance status
  • The report resets daily at 7:00 AM, adding new permit records while retaining closed or suspended permits until the next day's reset. This ensures all permits opened during the day are captured for reference.
Sheet 2: Layout
  • A Google Maps site screenshot was used as a background.
  • 87 subsections were overlaid on the map as interactive boxes.
  • Each box displays the total number of permits in that subsection.
  • The box color reflects the most critical permit in the area for easy risk assessment.
  • Permit issuers and supervisors can quickly:
    • Identify SIMOPS areas requiring additional safety controls.
    • Decide whether new activities can start or if existing permits must be closed first.
    • Recognize high-density work zones for enhanced supervision.
  • The map updates in real-time as permits are added, suspended or closed.

Reporting

  • The two Excel sheets are linked to PowerPoint for real-time updates.
  • The PowerPoint presentation includes:
    • One slide summarizing plant live activities.
    • Four slides detailing all permit data with statistics.
