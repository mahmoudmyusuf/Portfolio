Tokyo Olympics Medal Analysis Dashboard

This Power BI dashboard provides a comprehensive analysis of medal distribution, rankings, and country performance in the Tokyo Olympics. The project involves data cleaning, transformation, and visualization to generate meaningful insights.

📌 Resources

  • Data Sources:
    • Tokyo Olympic Data.xlsx – Contains medal statistics and rankings.
    • Country-region-table.xlsx – Manually compiled data for country mapping.

🛠️ Tools & Technologies

  • Power BI for Data Modeling & Visualization
  • Power Query for Data Cleaning & Transformation
  • DAX (Data Analysis Expressions) for Advanced Calculations

🔄 Data Cleaning & Preprocessing

  • Medals Data:
    • Removed unnecessary rows and promoted headers.
    • Trimmed text and changed data types for consistency.
    • Kept only relevant columns: "Team/NOC", "Gold", "Silver", "Bronze".
    • Created new columns to categorize medal winners.
    • Unpivoted medal columns for better filtering and visualization.
    • Removed rows where the number of medals was zero.
  • Medals Ranking:
    • Created a ranking table with "Rank", "Team/NOC", "Total", "Rank by Total".
    • Renamed "Rank" to "Rank by Gold" (ranking based on gold, then silver, then bronze).
    • Unpivoted ranking data for slicer functionality in the dashboard.
  • Country-Region Mapping:
    • Compiled a country-region mapping table for geographical visualization.
    • Corrected country names to align with standard Power BI mapping conventions (e.g., "Great Britain" → "United Kingdom").
    • Validated the dataset to ensure error-free integration in Power BI.

📊 Data Model & Relationships

  • Designed an Entity Relationship Diagram (ERD) to establish key relationships.
  • Linked medal data with country-region mapping for accurate geographical insights.
  • Created a structured model to facilitate dynamic filtering and ranking.

🔍 Key Analysis Questions

The project aims to answer critical questions related to Olympic medal trends:

  • How many total medals and winners were recorded?
  • What is the distribution of medal types across different countries?
  • How do regions and countries compare in terms of medal counts?
  • Which teams ranked highest based on gold medals and total medals?

📈 Dashboard Structure & Insights

The Power BI dashboard includes several key insights, powered by DAX calculations and interactive visualizations:

  • 🏅 Total Medal Overview – Displays the total count of medals won.
  • 📊 Medal Distribution – Shows medal distribution per winner and type.
  • 📌 Percentage Analysis – Calculates medal share percentages when slicers are applied.
  • 📊 Ranking Tables – Provides rankings based on gold and total medals.
  • 🗺️ Map Visualization – Geographical representation of medal distribution across regions.
