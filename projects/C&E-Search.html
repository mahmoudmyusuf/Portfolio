Project Overview

The Cause & Effect Search Tool is designed to simplify the process of identifying relationships between plant indicator signals (Causes) and final elements (Effects). It provides an efficient search mechanism, reducing manual efforts in analyzing signal interactions.

Cause & Effect Chart is a matrix where:

  • Empty cells indicate no relationship between the Cause and the Effect.
  • The letter X signifies an immediate effect initiation.
  • Other values like P (Permission required) or T20 (Effect after 20 minutes) provide additional control logic.

Data Cleaning & Preprocessing

  • Converted PDF files into Excel, ensuring each page was saved in a separate sheet.
  • Identified and fixed discrepancies:
    • Some empty cells in Excel contained data in the original PDFs.
    • Detected improper data, such as repeated "X" values across cells.
    • Removed leading/trailing spaces from text fields.
  • Verified and corrected "X" values using conditional formatting to highlight anomalies.
  • Standardized descriptions for better readability and consistency.
  • Consolidated all Causes into a single column and all Effects into another.

Data Integration Approach

Two approaches were considered for integrating the dataset:

  • Single Table Method: Merging all data into one table for easier filtering (previously used but found inefficient for updates).
  • Multi-Table Method (Chosen Approach): Retained individual page data while using the INDIRECT function for seamless cross-sheet filtering.

Cause & Effect Search Tool Features

  • Created a dedicated Search Page in Excel.
  • Implemented a dropdown menu to select Cause or Effect.
  • Developed an input field for partial signal/description searches.
  • Implemented an automated dropdown for filtered search results.
  • Upon selection, the tool displays:
    • Page number and index for reference.
    • Signal settings, ranges, notes, and voting details.
    • Associated causes if a tag is an effect, and vice versa.
    • Handling of duplicate tags with different settings via an additional dropdown menu.
    • Wildcard (*) support for flexible searches.

Impact & Benefits

  • Reduced search time by 80% for operational teams during work permit activities and signal isolation.
  • Enabled quick lookup of trip signals during upset conditions.
  • Integrated into DCS HMI for direct access by control room operators.
  • Eliminated the need for manual PDF searches, improving response efficiency.
