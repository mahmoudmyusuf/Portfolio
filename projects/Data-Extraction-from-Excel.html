Project Overview

Many companies and individuals still rely on Excel Workbooks for daily reporting rather than structured databases. Extracting data manually by copying and pasting from multiple sheets is time-consuming and inefficient. Since these reports follow a uniform structure, an automated extraction tool was developed to streamline the process.

Resources

  • Any Excel reports with a uniform structure (initial case: daily reports).

Tools & Technologies

  • Microsoft Excel
  • Excel VBA (Visual Basic for Applications)

Data Integration Approach

A custom Excel Workbook with VBA code (Extract.xlsm) was developed to automate data extraction. The tool runs either on-demand (button click) or at specific conditions. The VBA solution was refined through three trials:

This version allows users to define key inputs for extracting data efficiently:

  • Folder path: Location of target Excel workbooks (default: same folder as Extract.xlsm).
  • Target Sheet: Defined by name, sheet number, or defaults to the first sheet.
  • Target Cell Addresses: Manually specified in a defined row in Extract.xlsm.
  • Repeat Option: (Yes/No) ‚Äì Extract all data or only new records.
  • Highlight Option: (Yes/No) ‚Äì Highlight new data in yellow or disable highlighting.
  • The VBA code loops through all workbooks in the folder and extracts data based on the specified options.

üìå Second VBA Trial

This version enhances flexibility by leveraging direct cell links instead of predefined targets:

  • Folder path: Location of target Excel workbooks.
  • Copy & Paste Link: Directly paste cell links into a defined row in Extract.xlsm.
  • When executed, the script updates the links, retrieves new data, and appends it to the last row.

üöÄ Benefit: Enables data extraction from any location in target workbooks, eliminating the sheet limitation.

üìå Third VBA Trial

This version expands on the second approach by allowing data extraction from sub-folders using smart filtering:

  • Folder path: Includes all sub-folders.
  • Copy & Paste Link: Directly paste required cell links into a defined row.
  • Target Folder Name: Filter by keywords (Starts with / Ends with / Contains).
  • Target Workbook Name: Filter by keywords (Starts with / Ends with / Contains).
  • Retrieves data from all reports at once instead of running it multiple times for each sub-folder.

üöÄ Benefit: Automatically extracts data from hierarchical folder structures (e.g., monthly reports stored under year folders).
